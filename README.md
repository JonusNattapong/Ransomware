# Cassandra-Ransomware

## Description

This repository contains a Rust-based implementation of ransomware for educational and research purposes only. It demonstrates concepts in cryptography, file system traversal, and persistence mechanisms. **Warning: This code is for learning purposes and should not be used for malicious activities. Always ensure compliance with legal and ethical standards.**

## Goals

- Demonstrate encryption techniques using modern cryptographic libraries.
- Explore file system operations and traversal.
- Understand persistence and ransom note generation.
- Provide a basis for security research and defensive programming.

## Features

- File encryption using ChaCha20Poly1305.
- Recursive directory traversal.
- Persistence mechanisms.
- Ransom note generation.

## Requirements

- Rust 1.70 or later
- Cargo

## Build

```bash
cargo build --release
```

## Usage

**Do not run this on production systems or without explicit permission.**

```bash
cargo run
```
